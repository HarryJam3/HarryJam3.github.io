Office Area

Chrome

  • 基于本人的谷歌账号将一直沿用，故暂不讨论其插件失效等话题。
    • Carrot 用于 Codeforces 的赛中or赛后预测即时分数使用。偶尔失效
    • CF Analytics 用于在个人 Codeforces 主页生成生涯做题扇形统计图，更直观反应训练内容
    • Competitive Companion 用于快速远程抓取某场比赛中的某道题的样例映射到本地 Sublime Text
    • Enhanced GitHub 自动显示每个活动分支中的每个文件大小，显示每个单独文件的下载链接，将文件的内容直接复制到剪贴板，下载文件内容.etc
    • HackBar 简单抓包，允许用户对网站执行简单自定义HTTP请求，提供简单参数注入
    • JSON-handle 树形图直观美观展示 JSON 数据，自动渲染实时编辑
    • Mathpix Snip 可以OCR技术识别数学公式，并将其转换为LaTeX代码
    • Proxy SwitchyOmega 快速切换和管理浏览器代理设置
    • Stylish 网站美化的一个插件
    • Twitter Downloader HD 在推特/X上方便下载视频的一个插件_(:з」∠)_
    • Wappalyzer 主要用于识别网站所使用的技术栈
    • 哔哩哔哩Downloader 支持bilibili.com的视频一键下载
    • 沉浸式翻译 特别好用的网页翻译插件以及支持中英文字幕
    • 篡改猴/Tampermonkey 脚本管理工具，必备脚本如下
      • Codeforces Better!
      • Atcoder Better!
      • OCS 网课助手，刷网课常用
      • FCQ 全网答题器，水作业考试常用

即时通讯工具

  • QQNT，微信，Telegram，Discord，Oopz，飞书

文件全局搜索工具

Everything

  • 开机自启动默认扫盘，影响启动速度。但是在 Windows NTFS 和 ReFS 文件系统中胜在搜索效率极高，响应快。

    地址：https://www.voidtools.com/zh-cn/downloads/

Utools

  • 内置 Everything ，特定需求下触发。Alt+Space 触发搜索方便快捷，无需充值会员。

  • 包含各种小工具 eg.内网IP直接复制到剪切板，便于寻找一些设置项

    下载地址：https://www.u-tools.cn/

磁盘空间分析软件

SpaceSniffer

  • 主要用于磁盘空间的可视化管理和优化

    下载地址：https://www.fosshub.com/SpaceSniffer.html

硬件自查工具

CPU-Z

  • 提供关于计算机硬件和系统配置的详细信息，包括处理器、主板、内存、显卡等关键组件的技术规格和性能数据

    下载地址：https://www.cpuid.com/softwares/cpu-z.html/

GPU-Z

  • 显卡信息监测工具，能够提供详尽的显卡硬件信息以及实时监控显卡的运行状态

    下载地址：https://www.techpowerup.com/download/techpowerup-gpu-z/

网络代理工具

Clash for Windows

  • 帮助用户管理和配置网络代理服务，实现网络流量的转发控制，称为所谓“魔法”

    下载地址：https://clashforwindows.org/（非官方）

V2ray

  • 上手相对容易但是配置写的 JSON 比Clash用的 YAML相对难懂 下载地址：https://github.com/v2ray/v2ray-core

大学摸鱼工具

FCQ全网答题器

  • 快速且隐秘地完成答题刷课等功能，缺点是要钱买积分。肯定有更好的但是用习惯了懒得找 下载地址：https://tcb-w644nfbyxrttaih-2cpr71dbf4b7-1304481250.tcloudbaseapp.com/#/pages/index/index?orgin=group

Gaming Area

Steam

  • 最大的 PC 游戏数字发行平台

    下载地址：https://store.steampowered.com/about/

网站工具合集

似乎偏题了但是不重要

最近在学习剪辑视频的过程中，因为以往打 CTF 的比赛经验会有很多方便在网页端处理材料的工具站，事实证明确实如此但是在自己的收藏夹里发现了大量曾经觉得好用爱用的工具站但是现在觉得陌生或生疏。遂整理之。

ACM相关

AtCoder Problem

  • https://kenkoooo.com/atcoder/#/table/
  • 类似的有 CFTracker 地址为 https://cftracker.netlify.app/contests
  • 一个实时集合了每一届 AtCoder / Codeforces 比赛的网站，简化了查询题目和整理补题的操作，良心好站

Graph Editor

  • https://csacademy.com/app/graph_editor/
  • 方便将图论的样例可视化以及生成图片，如果你没有那么熟悉矢量图 paf 的绘制的话，使用它出题也不失为良策

ecpp + C++ - Try It Online

  • https://tio.run/#ecpp-cpp
  • 一个可以在线编译 cpp 并且方便自定义输入输出的网站，江湖救急必备。

OJ Analyzer

  • https://ojhunt.com/statistics
  • 它可以统计你在各大 OnlineJudge 上的刷题数，形成汇总数据。

cppreference.com

  • https://zh.cppreference.com/
  • 提供了 C++ 语言、标准库和其他常用库的详细文档，已经包含了 C++20 的内容，并且正在更新 C++23

CLIST

  • https://clist.by/
  • 提供了各种编程竞赛、CTF 比赛等活动的日程和信息。可以在这里找到即将开始或正在进行的比赛，并查看它们的结束时间、持续时长和相关链接

Board-XCPCIO

  • https://board.xcpcio.com/
  • 集合了大多数 ICPC 和 CCPC 的大型赛事的比赛榜单，包含各类邀请赛和各省省赛，支持筛选和导出

OIerDb NG

  • https://oier.baoshuo.dev/
  • 一个信息学竞赛选手获奖记录并对学校进行排名的数据库。能够在这个网站上查询OI选手的获奖记录

Dashboard-Challenge

  • https://challonge.com/zh_CN/dashboard
  • 一个管理与创建赛事流程图以及通过树状图的形式可视化预测对阵的看板

CTF相关

CyberChef

  • https://gchq.github.io/CyberChef/
  • 英国政府通信总部开发的开源工具，被誉为“网络瑞士军刀”。支持多种编码方式，提供AES、DES、Blowfish等多种加密算法。支持常见的数据压缩格式。能够计算MD5、SHA系列等多种哈希值。可以生成和解析二进制和十六进制转储。能够解析IPv6地址、X.509证书等。

Vigenere Solver

  • https://www.guballa.de/vigenere-solver
  • 在不知道密钥的情况下破解维吉尼亚密码，除了经典变体外还支持Beaufort密码和Autokey密码

factordb

  • https://factordb.com/
  • 主要用于存储和检索大整数的因子分解信息

quipqiup

  • https://quipqiup.com/
  • 在线密码文求解器，专门用于解决简单替换或词频分析谜题

Sage Cell Server

  • https://sagecell.sagemath.org/
  • 一个很好用的网络接口，提供对开源数学软件系统SageMath的访问，并允许将Sage计算嵌入到任何网页

Aperi'Solve

  • https://www.aperisolve.com/
  • 专门进行图像的层分析和深度隐写分析的在线工具，支持多种图像格式

HexEd.it

  • https://hexed.it/
  • 基于浏览器的在线和离线十六进制编辑器

ciphey.online

  • https://ww25.discord.ciphey.online/
  • 在线密码破解和密码分析工具，自动化尝试破解

游戏相关

Flux HvH

  • https://cshvh.cn/
  • 一个hvh服务器管理云平台以及各Rage卡网直充

天赐小铺

  • https://buy.tcstore.top/?categoryId=-10
  • 出售各类游戏小道具以及辅助工具

仓鼠杂货铺

  • https://cs24k.top/
  • Hvh社区获取参数、设备以及外设的集合，也包含其他游戏

FPSGO畅玩cs1.6

  • https://fpsgo.net/
  • 可以在网页上畅玩cs1.6，半条命以及SUROI等怀旧游戏

ProSetting

  • https://prosettings.net/games/cs2/
  • 集合了很多FPS职业玩家的灵敏度，持枪视角以及外设，作为参考资料站

NameMC

  • https://namemc.com/
  • 一个收集了大量社区玩家自制MC皮肤的成品网站可以在里面挑选自己的MC皮肤（虽然广告较多

Preaim练枪服

  • https://www.yuque.com/u138462/preaim
  • 一个人使用的bot会开枪会peek的练枪服，也支持练习道具，缺点是花钱

BetterGI

  • https://bgi.huiyadan.com/
  • 基于计算机视觉技术，集成ONNXRuntime，YOLOv8识别、多种OCR引擎的项目。意图是优化玩家游戏体验
