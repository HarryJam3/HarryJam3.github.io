关于我的电脑的一些配置以及工具备份（自用向
2024-11-28 22:34   label 软硬件配置规范
  1. 1. Chrome

Chrome

  • 基于本人的谷歌账号将一直沿用，故暂不讨论其插件失效等话题。
    • Carrot 用于 \(Codeforces\) 的赛中or赛后预测即时分数使用。偶尔失效
    • CF Analytics 用于在个人 \(Codeforces\) 主页生成生涯做题扇形统计图，更直观反应训练内容
    • Competitive Companion 用于快速远程抓取某场比赛中的某道题的样例映射到本地 \(Sublime Text\)
    • Enhanced GitHub 自动显示每个活动分支中的每个文件大小，显示每个单独文件的下载链接，将文件的内容直接复制到剪贴板，下载文件内容.etc
    • HackBar 简单抓包，允许用户对网站执行简单自定义HTTP请求，提供简单参数注入
    • JSON-handle 树形图直观美观展示 JSON 数据，自动渲染实时编辑
    • Mathpix Snip 可以OCR技术识别数学公式，并将其转换为LaTeX代码
    • Proxy SwitchyOmega 快速切换和管理浏览器代理设置
    • Stylish 网站美化的一个插件
    • Twitter Downloader HD 在推特/X上方便下载视频的一个插件_(:з」∠)_
    • Wappalyzer 主要用于识别网站所使用的技术栈
    • 沉浸式翻译 特别好用的网页翻译插件以及支持中英文字幕
    • 篡改猴/Tampermonkey 脚本管理工具
