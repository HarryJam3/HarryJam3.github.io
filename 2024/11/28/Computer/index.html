Office Area

Chrome

  • 基于本人的谷歌账号将一直沿用，故暂不讨论其插件失效等话题。
    • Carrot 用于 Codeforces 的赛中or赛后预测即时分数使用。偶尔失效
    • CF Analytics 用于在个人 Codeforces 主页生成生涯做题扇形统计图，更直观反应训练内容
    • Competitive Companion 用于快速远程抓取某场比赛中的某道题的样例映射到本地 Sublime Text
    • Enhanced GitHub 自动显示每个活动分支中的每个文件大小，显示每个单独文件的下载链接，将文件的内容直接复制到剪贴板，下载文件内容.etc
    • HackBar 简单抓包，允许用户对网站执行简单自定义HTTP请求，提供简单参数注入
    • JSON-handle 树形图直观美观展示 JSON 数据，自动渲染实时编辑
    • Mathpix Snip 可以OCR技术识别数学公式，并将其转换为LaTeX代码
    • Proxy SwitchyOmega 快速切换和管理浏览器代理设置
    • Stylish 网站美化的一个插件
    • Twitter Downloader HD 在推特/X上方便下载视频的一个插件_(:з」∠)_
    • Wappalyzer 主要用于识别网站所使用的技术栈
    • 沉浸式翻译 特别好用的网页翻译插件以及支持中英文字幕
    • 篡改猴/Tampermonkey 脚本管理工具，必备脚本如下
      • Codeforces Better!
      • Atcoder Better!
      • OCS 网课助手，刷网课常用
      • FCQ 全网答题器，水作业考试常用

即时通讯工具

  • QQNT，微信，Telegram，Discord，Oopz，飞书

文件全局搜索工具

Everything

  • 开机自启动默认扫盘，影响启动速度。但是在 Windows NTFS 和 ReFS 文件系统中胜在搜索效率极高，响应快。

    地址：https://www.voidtools.com/zh-cn/downloads/

Utools

  • 内置 Everything ，特定需求下触发。Alt+Space 触发搜索方便快捷，无需充值会员。

  • 包含各种小工具 eg.内网IP直接复制到剪切板，便于寻找一些设置项

    下载地址：https://www.u-tools.cn/

磁盘空间分析软件

SpaceSniffer

  • 主要用于磁盘空间的可视化管理和优化

    下载地址：https://www.fosshub.com/SpaceSniffer.html

硬件自查工具

CPU-Z

  • 提供关于计算机硬件和系统配置的详细信息，包括处理器、主板、内存、显卡等关键组件的技术规格和性能数据

    下载地址：https://www.cpuid.com/softwares/cpu-z.html/

GPU-Z

  • 显卡信息监测工具，能够提供详尽的显卡硬件信息以及实时监控显卡的运行状态

    下载地址：https://www.techpowerup.com/download/techpowerup-gpu-z/

Gaming Area

Steam

  • 最大的 PC 游戏数字发行平台

    下载地址：https://store.steampowered.com/about/
